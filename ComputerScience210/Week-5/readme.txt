Guessing Game (Counting Game) -- Classic number guessing game. Utilizes scanner with standard i/o and while/if/else loops. Math.random to generate
                                 random number which is stored into the answer variable, and while/if/else lops with greater/lesser than operators
                                 to continue asking user to enter a number -- with specific given hints -- until the user gets the correct number.
                                 Number of guesses is incremented at the end of each loop, and is printed to user once they succeed. Capable of
                                 playing multiple rounds, which is also tracked, if the user so decides to.
